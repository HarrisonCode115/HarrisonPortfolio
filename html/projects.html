AI-Chat

AI Chat brings multiple popular LLM's together into one place. Different models are good at different things, and sometimes it's quite a pain switching between the different models. On AI Chat, it is a concept that brings this all together, no constant tab changes and new conversations. Simple, quick change. Currently implemented is OpenAI's Chat GPT and Anthropic's Claude. If you want to see the source code please click here!

MyneCraft

MyneCraft is a game where you can break and place blocks and explore the world. The reason for this game is to learn and improve my skills using AI IDEs. I made use of Cursor for this project. If you want to play it please click here or use the option on the navigation bar!

Formula tracker is a React-based application to learn the basics of React. It tracks championship standings in real time using an API. You can view the repository here!
